Windows Installer – 64-bit

 

2.       Install GIT
https://git-scm.com/download/win
verify installation (under Git Bash, or go windows command )
$ git version
$ node –v
$ npm –v

3.       Install HEXO (go windows command, under c:\)
$ npm install -g hexo-cli
$ hexo –v

4.       Configure Github
create a new repository (repository name=”accountname.github.io”)

5.       Configure SSH Key
a. global configuration (copy below command in Windows command)
$ git config --global user.name "myRedeemer"
$ git config --global user.email "lorence.liu@gmail.com"

b. create SSH Key
$ ssh-keygen -t rsa -b 4096 -C "lorence.liu@gmail.com"

c. check SSH Key (under Git Bash)
$ cd ~/.ssh
$ cat ~/.ssh/id_rsa.pub

d. copy SSH Key

e. Go Github, Add SSH Key (Settings ->SSH & GPG Keys -> New SSH key ->past)

f. Authorize SSH Key
$ ssh -T git@github.com


6.  Initialize HEXO (under Windows command)
$ hexo init <folder> (folder name :mbg )
$ cd <folder>
$ npm install

7. test HEXO$ hexo generate
$ hexo server
$ npm install
http://localhost:4000

8.       Install deployment plug-in
$ npm install hexo-deployer-git --save  

9.       Configure HEXO _config.yml
